This page demonstrates the embeddable chat widget script running in an external site context.

A floating chat bubble has been injected in the bottom-right corner of the viewport.

To integrate this widget on any site, copy the following snippet:

<script 
  src="http://localhost:3000/widget.js" 
  data-bot-id="ca111111-1111-4111-8111-111111111111">
</script>
